# OIBSIP Python Projects

This repository contains various Python projects for the OIBSIP internship program.  

## BMI Calculator

The BMI Calculator is a project that provides an easy way to calculate body mass index and understand your body weight relative to your height.

## Usage

To use the BMI Calculator, run the script and input your height and weight. The calculator will provide you with your BMI and weight classification.
